# Conference Mic Control — пульт управления конференц-микрофонами

Веб-приложение для управления микрофонами в переговорной/конференц-зале через два устройства:

- **VIS-DCP1000** — включение/выключение микрофонов (TCP, бинарный протокол);
- **Tendzone Ruby T20** (или совместимый DSP) — регулировка громкости (UDP, текстовый протокол).

Интерфейс — схема зала с микрофонами по периметру стола (нажатие = вкл/выкл) и отдельная
вкладка-микшер для громкости. Состояние синхронизируется между устройствами в реальном времени.

Приложение кроссплатформенное (Windows, Linux), **без внешних зависимостей** — нужен только
Node.js Для планшетов есть Android-киоск (WebView) в папке `kiosk`.

> Все адреса устройств и каналы задаются в веб-интерфейсе на странице «Настройки».
> В репозитории нет привязки к конкретной сети — настраивается под свою инсталляцию.

---

## Возможности

- Схема зала: овальный стол, микрофоны по периметру, нумерация настраивается.
- Вкл/выкл каждого микрофона и всех сразу.
- Громкость и mute каждого канала (вкладка «Микшер»).
- Обратная связь о состоянии микрофонов от DCP1000.
- Веб-настройка адресов устройств и привязки каналов (без правки файлов).
- Адаптация под сенсорный планшет (1280x800, альбомная).

## Быстрый старт (сервер)

Нужен Node.js 18+.

    # Linux 
    ./start.sh

    # Windows
    start.bat

Откройте http://localhost:8080, зайдите в Настройки, введите адреса устройств
(DCP1000 и Tendzone) и привязку микрофонов к каналам.

С других устройств в той же сети — http://<IP-сервера>:8080

Подробные инструкции:
- Windows (в т.ч. офлайн): ЗАПУСК_СЕРВЕРА_WINDOWS_ОФЛАЙН.md
- РЕД ОС / Linux: УСТАНОВКА_REDOS.md
- Планшет (Android-киоск): УСТАНОВКА_НА_ПЛАНШЕТ.md и папка android-kiosk/

## Протоколы устройств

- DCP1000 — TCP, кадры [0x00,0x06,0x00,0x11, V, N, 0xFC,0xFC]
  (V: вкл=0x00, выкл=0x01; N — номер микрофона); обратная связь [0xFE,0x11, V, N, 0xFC].
- Tendzone — UDP, строки SET,0,INPUT0,GAIN,N,N,<значение>. и SET,0,INPUT0,MUTE,N,N,<0|1>.

Порты по умолчанию (10166 для DCP1000, 5557 для Tendzone) — стандартные для этих устройств,
меняются в настройках.

## Настройка под своё оборудование

Всё через веб-интерфейс (страница Настройки), файлы править не нужно:
- IP и порт DCP1000;
- IP и порт Tendzone, диапазон громкости;
- таблица микрофонов: имя, канал DCP (вкл/выкл), канал TZ (громкость), группа.

Настройки сохраняются в data/config.json (этот файл в .gitignore — у каждого свой).
